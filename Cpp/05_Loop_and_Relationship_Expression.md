# 루프와 관계 표현식

- C++에서는 **for, while, do while**이라는 세 종류의 루프를 제공한다.

- 루프는 조건 검사 표현식이 **ture 또는 0이 아닌 값으로 평가**되는 동안 구문들의 집합을 **반복하여 수행**한다.

- 조건 검사 표현식이 **false 또는 0**으로 평가되면 **루프를 종료**한다.

- **for 루프와 while 루프**는 루프 몸체에 있는 구문들을 **실행하기 전**에 먼저 조건을 평가하는 **진입 조건 루프**이다.

- **do while 루프**는 루프 몸체의 구문들을 **먼저 실행**한 후에 **조건 검사 표현식을 평가**한다.

- #### 루프의 조건 검사 표현식

  - 두 개의 값을 비교하는 **관계 표현식**이 자주 사용된다.
  - 관계 표현식은 6개의 관계 연산자 **<, <=, ==, >= !=** 중에서 어느 하나를 사용한다.
  - 관계 표현식은 **bool값 true 또는 false**로 평가된다.

- #### istream 클래스

  - ```cpp
    cin >> ch;
    ```

    - 입력 문자를 ch에 저장한다.
    - **빈칸 문자나 탭 문자, 개행 문자 등은 무시하고 건너뛴다.**

  - ```cpp
    cin.get(ch);
    ```

    - 입력 문자를 읽어들여 그 값이 **무엇이든** 간에 ch에 저장한다.

    - cin.get() 멤버 함수의 호출은 빈칸 문자나 개행 문자, 탭 문자도 포함하여 입력받은 문자를 리턴한다.

    - 사용 예

      ```cpp
      ch = cin.get();
      ```

  - EOF(End Of File)

    - 파일의 끝을 표현하기 위한 상수
    - -1 값을 지닌다.
    - fgetc, getchar 함수가 파일의 끝에 도달하는 경우 반환한다.

  - **cin.get(char)** 멤버 함수의 호출은 **EOF**를 만나면 **bool값으로 변환한 false를 리턴**한다.

  - **cin.get()** 멤버 함수의 호출은 **EOF**를 만나면 **iostream 파일에 정의되어 있는 EOF를 리턴**한다.

- 중첩 루프란 루프 안에 있는 다른 루프가 들어 있는 것이다.
