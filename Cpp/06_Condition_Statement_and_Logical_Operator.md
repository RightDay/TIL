# 분기 구문과 논리 연산자

* C++는 **if 구문, if else 구문, switch 구문**을 제공한다.

* ### if 구문

  * 단일 구문이나 구문 블록을 **조건에 따라 수행**하게 만든다.
  * 특정한 조건이 만족될 때에만 단일 구문이나 구문 블록을 수행한다.



* ### if else 구문

  * 두 개의 단일 구문이나 구문 블록 중에서 **어느 것을 실행할 것인지 선택**하게 만든다.
  * 여러 가지 선택 사항 중에서 선택하게 하려면 그와 같은 하나의 **if else 구문에 원하는 만큼의 if else를 추가**할 수 있다.



* ### switch 구문

  * 선택 사항의 목록에서 하나의 **특정한 경우를 선택하게 만든다.**



* ### 논리 연산자

  * if와 if else 구문에서 논리 연산자를 사용하여 관계 표현식들을 결합하거나 수정하여 더욱 정교한 조건 검사를 수행할 수 있다.

  * #### 논리합 OR 연산자(||)

    * 두 개의 표현식을 하나로 **결합한다.**

    * 두 개의 표현식 중에서 적어도 **어느 하나가 true이거나 0이 아니면 전체 표현식이 true가 된다.**

      * 그렇지 않은 경우에는 **전체 표현식이 false가 된다.**

    * || 연산자는 **시퀀스 포인트(sequence point\)**로 작용한다.

      * 좌변에 지시된 값의 변경은 **우변이 평가되기 전**에 일어난다.

      *  ```cpp
        i++ < 6 || i == j	// i = 10
        ```

        * i에 10이 있었다고 가정했을 때, j와의 비교가 이루어질 때 쯤이면 i의 값은 이미 11로 변해 있다.
        * 어느 하나의 표현식이 true이면 전체 논리 표현식이 true가 되므로, C++는 좌변의 표현식이 true일 경우에 **우변의 표현식은 평가하지 않는다.**
          * 세미콜론과 콤마 연산자도 시퀀스 포인트이다.

  

  * #### 논리곱 AND 연산자(&&)

    * 두 개의 표현식 **모두 true**일 때에만 전체 표현식이 true가 된다.

  

  * #### 논리부정 NOT 연산자(!)

    * ! 연산자는 뒤따르는 **표현식의 값을 반대로 만든다.**

    * 관계 연산자나 산술 연산자보다 **우선순위가 높다**

      * 어떤 표현식을 부정으로 만들기 위해서는 표현식을 **소괄호로 묶어야 한다.**

      * ```cpp
        !(x > 5)	//x가 5보다 크다의 부정
        !x > 5	//!x가 5보다 크다
        ```
