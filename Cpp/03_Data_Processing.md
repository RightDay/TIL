# 데이터 처리

* C++에 내장된 데이터형에는 기본형과 복합형이 있다.

* 기본형에는 정수를 표현할 수 있는 정수형과 소수부가 있는 수를 표현할 수 있는 소수점형이 있다.




* ## 간단한 변수

  * ### 변수 이름

    * C++에서는 의미를 쉽게 알 수 있는 변수 이름을 사용할 것을 권장한다.
    * 변수 이름을 짓는 규칙
        * 변수 이름에는 영문자, 숫자, 밑줄(_)만 사용할 수 있다.
        * 숫자를 변수 이름의 첫 문자로 사용할 수 없다.
        * 변수 이름에서 대문자와 소문자는 구별된다.
        * C++의 키워드는 변수 이름으로 사용할 수 없다.
        * 두 개의 밑줄 문자로 시작하는 이름이나, 밑줄 문자와 대문자로 시작하는 이름은, 그것을 사용하는 컴파일러와 리소스가 사용하기로 예약되어 있다.
        * 하나의 밑줄 문자로 시작하는 이름은, 그것을 사용하는 컴파일러와 리소스가 전역 식별자(global identifier)로 사용하기로 예약되어 있다.(컴파일 에러 X)
        * 변수 이름의 길이는 제한이 없으며, 변수 이름에 쓰인 모든 문자들이 유효하다. 그러나, 어떤 플랫폼은 고유의 길이 제한이 있다.

  

  * ### 정수형

    * 소수부가 없는 수이다.
    * 컴퓨터의 메모리 용량에 한계가 있어 정수들의 부분 집합만을 나타낼 수 있다.
    * C++는 여러 가지 정수형을 제공한다.

  

  * ### 비트와 바이트

    * #### 비트(bit)

      * 컴퓨터의 메모리를 구성하는 기본 단위는 비트이다.
        * ex) 전기 스위치가 꺼진 상태는 값 0, 켜진 상태는 값 1을 나타낸다.
      * 8비트 폭의 메모리는 256가지(2^8)의 비트 조합을 만들 수 있다.

    * #### 바이트(byte)

      * 8비트의 메모라 단위를 의미한다.
      * 바이트 단위로 컴퓨터의 메모리 크기를 나타낼 때, 1킬로바이트는 1024바이트이고 1메가바이트는 1024킬로바이트이다.
      * C++에서 바이트는 컴파일러의 기본 문자 세트를 수용할 수 있는 최소한의 연속된 비트들로 구성된다.
        * 바이트를 구성하는 비트들의 조합으로 나타낼 수 있는 값의 수가, 컴파일러가 사용할 수 있는 문자 수와 같거나 그보다 커야 한다.
        * ASCII와 EBCDIC 문자 세트를 사용하는 시스템에서 C++의 바이트는 8비트이다.
        * Unicode같이 더 큰 문자 세트를 사용하는 시스템에서는 C++의 바이트는 16비트 또는 32비트이다.

  

  * ### short, int, long, long long 정수형

    * short, int, long은 정수를 저장하는 데 사용하는 비트 수가 다르다.

    * 컴퓨터마다 데이터형이 동일한 폭을 갖고 있지 않아, C++는 데이터형들의 최소 크기만을 정하여 표준을 제공한다.

      * short형은 최소한 16비트 폭을 가진다.
      * int형은 최소한 short만큼 크다.
        * 컴퓨터마다 폭이 16, 24, 32비트가 될 수 있다.
      * long형은 최소한 32비트 폭을 가지며,  최소한 int만큼은 크다.
      * long long형은 최소한 64비트 폭을 가지며 최소한 long만큼은 크다.

    * #### climits에 정의되어 있는 기호 상수들

      * CHAR_BIT, SHRT_MAX, INT_MAX, INT_MIN, LONG_MAX 등

  

  * ### unsigned형

    * 음의 정수값을 저장할 수 없는 변형이다.

    * 변수에 저장할 수 있는 최대값을 늘릴 수 있다.

      * ex) short형 -32768에서 +32767까지의 범위를 unsigned short는 0부터 65535까지의 범위를 갖는다.

    * ```cpp
      unsigned short change;	//unsigned short형
      unsigned int rovert;	//unsigned int형
      ```

  

  * ### 정수형 정하기

    * 일반적으로 int형은 컴퓨터가 가장 효율적으로 처리하는 정수 형식이다.
    * 메모리 절약이 중요한 경우에는 int형 대신 short형을 사용한다.

  

  * ### 정수형 상수

    * 212, 1667과 같이 프로그램에 직접 써 넣는 정수를 말한다.

    * 10진수, 8진수, 16진수로 정수형 상수를 프로그램에 직접 작성할 수 있다.

    * ```cpp
      int chest = 42;		//10진 정수형 상수
      int waist = 0x42;	//16진 정수형 상수
      int inseam = 042;	//8진 정수형 상수
      ```

      * chest 값 : 42
      * waist 값 : 66
      * inseam = 34

    * 이러한 표기법들은 표기를 쉽게 하기 위한 것이다.

    * 어느 것으로 표기하더라도 컴퓨터 내부에서는 모두 동일한 2진수 값으로 처리된다.

    * 어떤 정수값을 16진수나 8진수 형태로 출력하려면 출력하라는 메시지를 cout에 전달하는 dec, hex, oct 조정자를 제공한다.(10진수가 디폴트 형태이다.)

      * ```cpp
        int a = 42;
        
        cout << a << " (10진수)" << endl;
        cout << hex;	//16진법으로 바꾸는 조정자
        cout << a << " (16진수)" << endl;
        cout oct;		//8진법으로 바꾸는 조정자
        cout << a << " (18진수)" << endl;
        ```

        * 42 (10진수)
        * 2a (16진수)
        * 52 (8진수)

  

  * ### C++가 상수의 데이터형을 결정하는 방법

    * C++는 특별한 이유가 없다면 정수형 상수를 모두 int형으로 저장한다.
    * 그러나 특정 데이터형을 의미하는 접미어를 상수에 붙였을 때와, 값이 너무 커서 int형으로 저장할 수 없을 때에는 그렇지 않다.
      * l이나 L은 long형을 의미, u나 U는 unsigned int형을 의미한다.

  

  * ### char형: 문자와 작은 정수

    * 문자와 숫자를 저장한다.
    
    * 문자를 수치 코드로 나타낸다.
    
    * 1바이트이다.
    
    * 가장 많이 사용하는 문자 세트는 ASCII이다.
    
    * #### 확장 char형: wchar_t(wide character type)
    
      * 프로그램이 1바이트(8비트)로 표현할 수 없는 문자 세트(일본어, 중국어, 한국어)를 처리해야 할 경우 사용한다.
    
      * 시스템에서 사용되는 가장 큰 확장 문자 세트를 나타낼 수 있다.
    
      * 확장 문자 상수나 확장 문자 문자열은 그 앞에 L을 붙여서 나타낸다.
    
        * ```cpp
          wchar_t bob = L'P';	//확장 문자 상수
          wcout << L"tall" << endl;	//확장 문자 문자열의 출력
          ```
  
  
  
  * ### bool형
  
    * 참이나 거짓 중 어느 한 가지 값을 가진다.
  
    * C++는 0이 아닌 값을 참, 0인 값을 거짓으로 해석한다.
  
    * ```cpp
      bool start = -100;	//start에 true가 대입된다.
      bool stop = 0;		//stop에 false가 대입된다.
      ```
