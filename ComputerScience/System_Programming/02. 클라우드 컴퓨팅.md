# 02. 클라우드 컴퓨팅

클라우드 컴퓨팅 서비스 활용

- 최신 기술에 익숙해진다는 장점
- 동일한 환경에서 사용이 가능하다.



### 클라우드 컴퓨팅 서비스

서버 환경을 미리 구축해놓고, 간단한 설정으로 바로 사용할 수 있도록 만든 서비스이다.

이전 : 미리 서버 컴퓨터를 구축

현재 : 클라우드 컴퓨팅 환경 사용

Amazon Web Service(AWS)



## 클라우드 컴퓨팅 설정

1. EC2 또는 인스턴스(서버) 생성
2. Elastic IP (탄력적 IP) 생성
   - IP 주소 : 128.12.10.1
   - 동적 IP : 관리가 힘들다.
   - 고정 IP를 붙여줘야 한다.
     - 인터넷 환경으로 접속하려면 언제든지 접속하기 위해

- 자기 PC(클라이언트)에서 EC2(서버) 접속
