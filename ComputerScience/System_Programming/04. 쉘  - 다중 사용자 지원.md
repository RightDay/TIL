# 04. 쉘  - 다중 사용자 지원

## 쉘 종류

쉘(shell) : 사용자와 하드웨어와 운영체제간 인터페이스

사용자의 명령을 해석해서, 커널에 명령을 요청해주는 역할이다.

관련된 시스템콜을 사용해서 프로그래밍이 작성되어 있다.



Bourne-Again Shell (bash) : GNU 프로젝트의 일환으로 개발됨, 리눅스 거의 디폴트이다.

Burne Shell (sh)

C Shell (csh)

Korn Shell (ksh) : 유닉스에서 가장 많이 사용 됨
